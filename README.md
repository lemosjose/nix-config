# Nix-Config 


Simple repository for storing my current nix configuration, i have a simple setup, a Full-AMD Desktop, a samsung laptop and an old hp laptop working as a local media server with services related to media consumption.

All hostnames are based on "Garras da Patrulha" characters.

## Organization 

#### Home
Stores configurations related to home manager, programs, shell stuff and everything that can and should be configured in home-manager 

#### hosts

Store system-level configurations. The config is modularized to make the editing and reading easier and changes are made more softly
